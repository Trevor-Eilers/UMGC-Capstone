# UMGC-Capstone Project
A browser-based multiplayer game where 2-4 players each manage a district within a shared city. Each player adjusts policy levers (like tax rates, zoning, and infrastructure investment) and after each round, the simulation processes what happens across all districts. Population shifts, businesses relocate, commuter patterns change, and tax revenue goes up or down depending on what every player decided. 

The problem we are trying to address is that many people do not have a good way to see how interconnected municipal policy decisions really are. What looks like a smart move for one district can cause problems across the whole region, and that's hard to understand just from reading about it. This game aims to make those dynamics something you experience.
