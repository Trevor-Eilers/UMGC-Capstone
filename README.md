# UMGC-Capstone Project
A browser-based multiplayer game where 2-4 players each manage a district within a shared city. Each player adjusts policy levers like tax rates, zoning, transit spending, housing density, and public services, and after each round a simulation engine processes what happens across all districts at once. Population shifts, businesses relocate, commute patterns change, and tax revenue goes up or down depending on what every player decided. The problem we're trying to address is that most people don't have a good way to see how interconnected municipal policy decisions really are. What looks like a smart move for one district can cause problems across the whole region, and that's hard to understand just from reading about it. A game makes that dynamic something you can actually experience. On the technical side, we're planning a Python backend for the simulation engine, a frontend built with React and Unity for the game interface and dashboards, multiplayer infrastructure for player coordination, and an AI layer using a large language model to give players strategic feedback after each round instead of just raw numbers.
